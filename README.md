# consumer-financial-complaints-analysis
Power BI dashboard analyzing 1M+ U.S. consumer financial complaints to identify product trends, response rates, and geographic complaint distribution.

Project Overview

This Power BI dashboard analyzes over 1 million U.S. consumer financial complaints to identify trends in product categories, company response behavior, and geographic distribution. The project demonstrates data cleaning, exploratory analysis, and interactive dashboard design using Power BI.

Key Insights

• Credit reporting services generate the largest share of complaints.
• Consumer complaints increased significantly after 2020.
• Most companies provide timely responses to complaints.
• Complaints are heavily concentrated in major U.S. population centers.

Tools Used

• Power BI
• Data Cleaning and Transformation
• Exploratory Data Analysis
• Interactive Dashboard Design
